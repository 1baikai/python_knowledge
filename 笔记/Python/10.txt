Python函数的参数传递
传递方式：
	位置传参
	    序列传参
	关键字传参
	    字典关键字传参、

位置传参：
	实际参数的对应关系与形式参数的对应关系是位置来依次对应的
示意：
	def mymin(a,b,c):
	    pass
	mymin(1,2,3)#123位置位子对应关系传给a,b,c
说明：
	实际参数和形式参数通过位置传递和匹配
	实际参数的个数必须与形式参数的个数相同、、
示例见1

序列传参：
	序列传参是指在函数调用过程中，用*将序列拆解后按位置进行传递的传参方式
说明：
	序列传参时，序列拆解的位置将与形参一一对应
	序列得到位置信息对应相应的参数位置
示例见2


关键字传参：
	关键字传参是指传参时，按着形参的名称给形参赋值
	实参的形参按名称进行匹配
示意：
	def myfun (a,b,c):
	    pass
	myfun(b=22,c=33,a=11)  #11->a  22->b c->33
	myfun(c=66,b=55,a=44)  #
说明：
	实参和形参按着形参名进行匹配，可以不按位置进行匹配
示例见3

字典关键字传参：
	是指实参为字典，将字典用**拆解后进行关键字传参的方式
说明：
	字典的键名和形参名一致
	字典的键名必须为字符串（且为标识符的命名方式）
	字典的键名要在形参存在
示例：4

函数的综合传参：
	函数的传参方式，再能确定形参能唯一匹配到想应实参的情况下可以任意组合
	函数的位置传参要先于关键字传参
说明：
位置传参和序列传参可以混合使用，且可以顺序颠倒
	如:
	myfun (100,*[200,300])
myfun(*[100,200],300)
myfun(*[100],200,*[300])
函数的关键字传参和字典关键字传参可以混合使用，且可以顺序颠倒、
	如：


myfun(c=300,**{'b':200,'a':100})
myfun(**{'b':200,'c':300},a=100)
位置传参和关键字传参可以混合使用，但位置传参要先于关键字传参：
	如：
	myfun（100，c=600,b=200）
	myfun(10,c=30,**{'b':20})

函数的缺省参数：
    语法：
	def函数名（形参名1=默认实参1，形参名2=默认实参2，...）:
	    语句块
    示例5・：
    说明：
	缺省参数必须自右至左依次存在，如果一个参数有缺省参数，则其右侧的所有参数都必须有缺省参数
	缺省参数可以有0个或者多个，甚至全部都有缺省参数数
    示例：
	def fn(a=10,b=10,c=10):
	    pass

函数形参的定义方式：
	位置形参
	星号元组形参
	命名关键字形参
	双星字典形参
位置形参
	def 函数名（形参名1，形参名2，....）：
	    语句块
	星号元组形参
	    语法：
		def  函数名（*元组形参名）：
		    语句块
	    作用L：收集多余的位置传参
	    说明：元组形参名一般命名为‘args’
	示例：6

命名关键字形参：
	语法：
	    def 函数名（*，命名关键字形参）：
		语句块
	或
	def  函数名（*args，命名关键字形参）：
	    语句块
	作用：
	    强制所有的参数收必须用关键字传参或字典关键字传参
示例：7

双星号字典形参
	语法：
	    def  函数名（**字典形参名）：
		语句块
	作用：
	    收集多余的关键字传参
	说明：
	字典形参名一般命名为kwaegs


函数的参数说明：
	位置形参，缺省参数 星号元组形参，双星号字典形参等可以混合使用
函数参数自左至右的顺序为：
	1 位置形参
	2、星号元组形参
	3、命名关键字形参
	4、双星号字典形参
示例
	def fn (a,b,*args,c,**kwargs):
		pass
	fn(100,200,300,400,d='D',e='E',c='C'
