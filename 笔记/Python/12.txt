map 函数
	map（func,*iterablse）翻一个可迭代对象，此可迭代对象用函数func对可迭代对象iterable中的每一个元素作为参数计算后得到新的数据

 实例：
            def power2(x):
                return x ** 2
            for x in map(power2,range(1,10)): 
                print(x)


          filter函数：(过滤)
            fliter（function,iterable）
            作用：
              筛选可迭代对象iterable中的数据，返回一个可迭代对象，此可
              迭代对象只返回iterable中符合条件的数据
              function将对iterable中提供的每个数据进行求布尔值，如果
              为True则保留,返回False则丢弃此数据
            实例：(此函数判断x是否为奇数，如果为奇数返回TRUE，否则返回False)
               def isodd(x):(如果返回False,返回的结果丢弃)
                   return x%2 == 1
               for x in fliter(isodd,range(10)):
                   print(x)

递归函数recursion
	递归是指函数直接或间接的调用自身
递归示例：
	函数直接调用自身
	def f():
	    f()  #直接调用自身
f()
print(递归完成)

函数间接调用自身
def fa（）：
    fb()
def fb()
    fa(()
fa()
print("递归完成")

递归说明：
	递归一定要控制递归的层数，当符合某一个条件时要终止递归调用几乎所用的递归都能用while循环来代替
递归的优缺点
	优点：可以把问题简单化，让思路会为清晰，代码更简洁
	缺点：递归因系统环境影响大，当递归深度太大时，可能会得到不可预知的结果
递归调用分为两个阶段：
	递推阶段：
		从原问题触发，按递归公式递推，从位置到已知，最总达到递归终止条件
	回归阶段：
		按递归终止条件求出结果，你想逐步带入递归公式，回归到原问题求解

什么是闭包
	闭包是指引用了此函数外部变量的函数
如果一个内嵌函数访问了外部嵌套函数的变量，则这个内嵌函数就是闭包
闭包必须满足三个条件：
	1、必须有一个内嵌函数
	2、内嵌函数必须阴影外部函数中的变量
	3、外部函数返回值必须是内嵌函数
