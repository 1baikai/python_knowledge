文件和目录相关的命令：
	pwd命令：
		作用：用于显示当前操作的路径（当前工作文件夹）

	ls命令：
		作用：显示指定目录的文件或者文件夹
	常用选项:
		-l列表显示文件的详细信息
		-a显示全部文件、文件夹
	示例：

		LS -l core
        -rw-------  1 tarena tarena 5943296 7月  28  2017 core
	  ^               ^      ^      ^          ^        ^
          文件权限      属主   属组  文件大小  最后修改时间 文件名	
	

	cd命令：
		作用：改变当前的工作目录。
		格式：
			cd /home/tarena    #进入到这个文件夹
			cd /               #进入到根目录
			cd ..              #返回上一级
			cd ~               # 返回主目录
	mkdir:
		作用：创建一个或者多个文件夹
		格式：
		
	rmdir:
		作用：删除一个或者多个文件夹
		说明：文件内必须为空才可以用rmdir删除
		示例：
			mkdir mydir1 mydir2
			rmdir mydir1 mydir2
	touch:
		作用：
			1、如果文件不存在，则创建一个空文件
			2、如果文件或者木兰已存在，则用系统时间更新他的修改时间。
		格式：touch 文件名
		示例：touch newfile
	rm :
		作用：删除文件或者文件夹
		常用选项：
			-r 递归删除文佳佳内的文件或文件夹
			-i 删除前给出提示（y/n）
			-f 强制删除，不给任何提示；
		示例“rm newfile”

Linux 下的路径：

	（一个路径的起点）
	路径分为俩种：
		绝对路径“以/字符开头的路径为绝对路径”
		相对路径：“不以/字符开头的路径为相对路径”
		    开始符号：
			文件/文件夹名称
			. 当前文件夹
			.. 上一级文件夹
			~用户目录

文件管理命令
gzip命令：
	作用：用zip压缩算法对文件进行压缩，生成压缩后的。gz文件
	格式
		gzip 文件名
gunzip：
	作用：对用zip压缩的.gz文件进行解压缩
	格式：gunzip 文件名
	示例：gzip mynote.txt  #将mynote.txt 压缩为mynote.txt.gz
		gunzip mynote.txt.gz	#将mynote.txt.gz 解压缩
					
tar 打包 解包
	作用：对文件或文件夹进行打包解包操作
	格式：tar  【选项】 文件名 【文件名或路径】
		-c	创建包
		-x 	解包
		-f  文件名 操作的文件名
		-v 	显示操作的文件细节
		-z  	用gizp或者gunzip对包进行压缩和解压缩
	示例：cd aid1807
		tar -czvf Linux.tar.gz #将Linux文件打印并压缩为.gz
		tar -xzvf Linux.tar.gz  #解包
		tar -xzvf “”.tar.gz -C （路径）  #解压到指定文件夹
